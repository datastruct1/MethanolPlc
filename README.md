基于AvaloniaUI的燃料供应系统PLC模拟器
后续引入Modbus通信

## 简介

本项目是一个基于AvaloniaUI的燃料供应系统PLC模拟器，用于模拟燃料供应系统的运行状态。
该模拟器可以模拟PLC的输入输出信号，并显示PLC的运行状态。

## 功能

- 模拟PLC的输入输出信号
- 显示PLC的运行状态

## 1.项目结构

- `src`：源代码目录
  - `Service`：服务类，包括PLC服务类和信号服务类
  - `Model`：模型类，包括PLC类和信号类
  - `View`：视图类，包括PLC视图和信号视图
  - `ViewModel`：视图模型类，包括PLC视图模型和信号视图模型

## 2.工艺流程模拟

-具有较为仿真的流程模拟，包括燃料供应系统的各个部分，如燃料泵、燃料罐、燃料输送管道等。-具有符合规范指南的故障情景，包括燃料泄漏，可燃气体浓度超标等。

## 3.PLC模拟

-具有较为仿真的PLC模拟，包括PLC的输入输出信号，以及PLC的运行状态。-具有符合规范指南的故障情景，包括PLC的输入信号异常，PLC的输出信号异常等。
